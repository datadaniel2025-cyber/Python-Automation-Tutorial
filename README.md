# Python Excel Automation Tutorial

This repository is a beginner-to-advanced Python tutorial focused on Excel automation, data cleaning, reporting, and data visualization.

## What You Will Learn

- Python basics
- Reading Excel files
- Cleaning messy data
- Filtering and sorting data
- Creating summary reports
- Creating charts
- Transferring data between Excel files
- Building simple desktop UI tools
- Merging multiple Excel files
- Creating automated Excel reports

## Tools Used

- Python
- pandas
- openpyxl
- matplotlib
- tkinter
